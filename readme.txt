IDE the project was made in: Intellij IDEA 2019.2.1
The version of Java used to build the project: 12
Working directory used to build the game: Root of the GitHub repo
How to run the game:
    1.
        - Import the project into Intellij
        - Mark the root folder as source root
        - And run World.main(),
    2.
        - Double click the jar file in the jar folder.
          Running the jar file requires Java 12+ to be installed.
